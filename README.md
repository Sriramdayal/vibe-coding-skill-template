A customized, out-of-the-box GitHub template designed to supercharge AI vibe-coding assistants (like Google Antigravity, Claude Code, Cursor, etc.) with strict rules for **Python-based codebases**.

When you equip your AI agent with this template, it will inherently understand how to write clean, Pythonic, performant, and type-safe code right from the first prompt.

---

## ✨ Why Use This Template?

- **Optimal DSA Practices**: Strict guidelines to prioritize `O(n log n)` or better time complexities using Python's powerful built-ins like `collections`, `itertools`, and `heapq`.
- **Robust SDLC Management**: Built-in guardrails for CI/CD, isolated environments (`uv`/`venv`), unit testing (`pytest`), and static analysis (`mypy`/`ruff`).
- **Prompting Excellence**: A prompt engineering cheat sheet tailored specifically to extract maximum performance from LLMs generating Python.
- **Ready-to-Use Boilerplate**: Pre-configured code templates featuring type hints, PEP-8 compliance, and structured docstrings.

---

## 🚀 Getting Started

1. **Create your Repo**: Click the **Use this template** button at the top of this repository to create your own project.
2. **Clone** the generated repository to your local machine.
3. **Launch your Agent**: Open the project folder in your AI-powered IDE or start your command-line agent.
4. **Invoke the Skill**: Instruct your assistant to load the skill:
   > `/skill vibe-coding-guidelines-python`
5. Watch as your agent automatically ingests the constraints and generates best-in-class Python architecture!
